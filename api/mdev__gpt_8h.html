Detailed Description

General Purpose Timer (GPT) Driver.

General Purpose Timer (GPT) driver provides mdev interface to use 4 general purpose hardware timers.

Usage

A typical GPT device usage scenario is as follows:

  1. Initialize the GPT driver using gpt_drv_init().
  2. Open the GPT device handle using gpt_drv_open(). Here caller must specify GPT number (from 0 to 3)-
  3. Use gpt_drv_set() to set desired interval in micro-seconds.
  4. Close the GPT device using gpt_drv_close() after its use. gpt_drv_close() call informs the GPT driver to release the resources that it is holding. It is always good practice to free the resources after use.
  5. gpt_drv_setcb() should be used to set GPT callback function. This callback will be called after interval specified in gpt_drv_set() until GPT is explicitly stopped using gpt_drv_stop().
  6. Start GPT operation using gpt_drv_start().
  7. Stop GPT operation using gpt_drv_stop().
    Note
    gpt_drv_stop must not be called from callback registered using gpt_drv_setcb()or from interrupt context.
    Code snippet:
    Following code demonstrates how to use GPT driver APIs to generate a timeout.
    static void gpt_cb(void)
    {
    process_gpt_timeout_cb();
    }
    {
    gpt_drv_init(timer_id <0-3>);
    .

    mdev_t* gpt_dev = gpt_drv_open(timer_id<0-3>);
    if (gpt_dev == NULL)

    gpt_drv_setcb(gpt_dev, gpt_cb);

    .
    .
    .
    gpt_drv_stop(gpt_dev);
    gpt_drv_close(gpt_dev);
    return;
    }

Enable PWM using GPT

This function configures the PWM functionality using the General Purpose Timer.

Note
The correct PinMux needs to be set using the PINMUX driver before calling this function.

The following parameters are set by default
GPT_CH_FUNC_PWM_EDGE - Pulse starts at edge of cycle
GPT_CH_WAVE_POL_POS - Waveform goes from high to low

Parameters
[in]devHandle to the GPT device returned by gpt_drv_open().
[in]chThe Channel to generate PWM waveform on.
[in]tonNumber of cycles of GPT the output will be On.
[in]toffNumber of cycles of GPT the output will be Off.
