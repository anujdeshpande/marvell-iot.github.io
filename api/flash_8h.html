Detailed Description

Flash Driver.

This is a wrapper driver which abstracts the internal flash driver (iflash) and the external flash driver (xflash) (if configured) into one.

Steps to use flash with mdev

  1. Flash driver initialization

Call flash_drv_init from your application initialization function. This will initialize and register both internal and external flash drivers with mdev interface.

  1. Open Flash device for use

This should be performed once before using Flash hardware. Here is example code to open mdev flash handler:

{
mdev_t *fl_dev;
fl_dev = flash_drv_open(FL_FTFS_DEV);
}

Note: The parameter must be either FL_INT or FL_EXT depending upon which flash you want to use.

  1. Data read/write on Flash

flash_drv_read() and flash_drv_write() can be used to read and write data to/from the Flash device. Please note that you should pass the handle returned by flash_drv_open() to the flash_drv_read() and flash_drv_write() functions.

  1. Erase Flash

flash_drv_erase() amd flash_drv_eraseall are used to erase the flash chip.

  1. Closing Flash device.

flash_drv_close() call is used to close the flash driver.
