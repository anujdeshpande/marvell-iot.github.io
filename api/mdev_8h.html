Detailed Description

Marvell Device Driver API.

This provides the Marvell Device abstraction for communicating with various devices.

Usage

Drivers typically register an mdev API using the mdev_register() call. The applications can use this device as follows:

  1. open the device using [driver name]_drv_open()
  2. perform read and write on the device using [driver name]_drv_read() and [driver name]_drv_write() respectively.
  3. close the device [driver name]_drv_close()
