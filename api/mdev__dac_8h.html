Detailed Description

Digital Analog Converter Driver.

DAC driver provides mdev interface to use Digital to Analog Converter unit/s. Follow the steps below to use DAC driver from applications

  1. Include Files

Include mdev.h and mdev_dac.h in your source file.

  1. DAC driver initialization

Call dac_drv_init from your application initialization function. This will initialize and register dac driver with the mdev interface.

  1. Open DAC device to use

dac_drv_open should be called once before using DAC hardware. Here you will specify DAC channel (A or B). The other parameters that are set by default are -
DAC_WAVE_NORMAL - No predefined waveform
DAC_RANGE_LARGE - Output Range Large
DAC_OUTPUT_PAD - Output is enable to pad
DAC_NON_TIMING_CORRELATED - Non timing corelated mode

If these defaults need to be changed they should be changed in dac_drv_open

{
mdev_t *dac_dev;
dac_dev = dac_drv_open(MDEV_DAC, DAC_CH_A);
}
  1. Set output of DAC

Call dac_drv_output to get the output of the mentioned dac. Max resolution is 10 bit.

  1. Closing a device.

A call to dac_drv_close informs the DAC driver to release the resources that it is holding.

  1. De-register the DAC driver

A call to dac_drv_deinit informs DAC driver to de-register the DAC driver.

int  value

Modify the default DAC configuration

This function allows user to over-ride the default DAC configuration. A call to thif function is not mandatory and should be used only if the user is aware of its impact.

This function should be used after dac_drv_init() but before dac_drv_open().

Only one paramater can be changed in a single call. Use this function multiple times if multiple parameters need to be changed. All un-modified parameters are set to their default values. The default values are as below: Config-Type | Default value waveMode = DAC_WAVE_NORMAL outMode = DAC_OUTPUT_PAD timingMode = DAC_NON_TIMING_CORRELATED

Parameters
[in]config_typeThe config option to be changed. Can be waveMode, outMode or timingMode.
[in]valueThe desired value for the parameter.
